# Enforcing-Mandatory-Fields

Description:
This project demonstrates how to enforce mandatory fields in a form so that users cannot submit the form without entering required information. It ensures that important data such as name, email, password, or other required inputs are not left empty. This improves data accuracy and prevents incomplete form submissions.

Objective:
The main objective of this project is to validate user input and make sure that all required fields are filled before the form is submitted.

Features:

Prevents form submission if required fields are empty

Displays validation messages to the user

Ensures accurate data entry

Simple and user-friendly form validation


Technologies Used:

HTML

CSS

JavaScript / PHP (for validation)


How It Works:

1. The user opens the form.


2. The user enters information in the input fields.


3. If any mandatory field is left empty, a validation message appears.


4. The form will only be submitted when all required fields are filled.



Example:
If a user tries to submit the form without entering their name or email, the system will show an alert or error message asking the user to fill the required fields.

Conclusion:
Enforcing mandatory fields is an important part of form validation. It helps ensure that users provide all necessary information before submitting the form, making the data collected more reliable and complete.
